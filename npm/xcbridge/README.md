# @aptove/xcbridge

Xcode bridge service for containerized iOS development.

This is the main package that automatically installs the correct binary for your platform.

## Installation

```bash
npm install @aptove/xcbridge
```

## Usage

```bash
# Run directly
npx @aptove/xcbridge --port 9090

# Or if installed globally
xcbridge --port 9090
```

## Supported Platforms

- macOS Apple Silicon (darwin-arm64)
- macOS Intel (darwin-x64)
